# Proxy

Um proxy é capaz de interceptar diversos tipos de operaçoes em um objeto alvo.

Existem métodos, chamados de trap, para diversos tipos de eventos relacionados a um objeto como:
- apply
- constructor
- defineProperty
- deleteProperty
- get
- getOwnPropertyDescriptor
- getPrototypeOf
- has
- isExtensible
- ownKeys
- preventExtensions
- set
- setPrototypeOf
